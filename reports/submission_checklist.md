# Final Submission Checklist

Use this checklist before final submission and viva.

- [ ] `project_report.pdf` exported and readable.
- [ ] `presentation.pdf` exported and readable.
- [ ] Tableau dashboard links verified in `tableau/dashboard_links.md`.
- [ ] KPI numbers in report match notebook outputs.
- [ ] Visual labels and units are consistent (currency, days, percentages).
- [ ] Contribution matrix completed and mapped to commits.
- [ ] Repository is clean (`git status` shows no unintended changes).
